# Sports Narrative Generation and Visualization

## Introduction

To create basketball match data analysis reports consisting of automatically generated narrative match summary 
in terms of basketball match data accompanying a data visualization system.

## Data

NBA match data. I am working on some basic box score data of NBA match
[Dallas Mavericks vs Los Angeles Lakers](https://global.espn.com/nba/boxscore?gameId=401160929) on December 1, 2019.

## Language - Python/JavaScript

### Python 

Text generation part

To start with, I had one Explorative Study to let four participants gave their summaries on NBA match in terms of box score data 
in order to get an initial overview of what aspects to consider while summarizing basketball match's box score data.

### JavaScript - D3.js

Visualization part
